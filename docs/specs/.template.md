## MC/DC ケース表(複合条件がある場合のみ)

複合条件 `a && b && c` に対する独立性検証ケース。各条件が独立して結果を決定することを最小ケース数で示す。

| # | a | b | c | 期待結果 | 備考 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 1 | T | T | T | T | 全真 |
| 2 | F | * | * | F | a が独立に結果決定 |
| 3 | T | F | * | F | b が独立に結果決定 |
| 4 | T | T | F | F | c が独立に結果決定 |

`*` は短絡評価により評価されない条件。
